# 比特币测试项目
    
## 项目介绍

该项目是使用bit-node框架搭建的比特币测试项目，该项目包含以下功能：

- createAddress 创建钱包
- createAddress 创建账户
- createAddress 创建地址
- transcation 创建交易
- detailAddress 查询交易
- detailAddress 查询余额
- detailAddress 查询区块
- detailAddress 查询地址
- detailAddress 查询账户
- detailAddress 查询钱包
